Sony TV Switch

This app allows the easy control and program switching via network for many Sony TVs. It can also be used as plugin for the TV Browser app to switch to programs from an electronic program guide (EPG).

Key features

  • Supports many 'smart' Sony TV models
  • Full featured remote control over network
  • Power on TV via Wake-on-LAN (WoL)
  • Shows program list from TV to easily search for and switch to programs from the app
  • Can be set-up for multiple Sony TVs
  • As TV Browser plugin: Switch to a TV program directly from the program guide (EPG)

Prepare your TV

  1. Switch on TV
  2. Enable remote start on your TV:
    [HOME] → [Settings] → [Network] → [Home Network Setup] → [Remote Start] → [On]
  3. Enable pre-shared key on your TV if possible (optional):
    [HOME] → [Settings] → [Network] → [Home Network Setup] → [IP Control] → [Authentication] → [Normal and Pre-Shared Key]
  4. Set pre-shared key (arbitrary string XXXX) on your TV if this option exists:
    [HOME] → [Settings] → [Network] → [Home Network Setup] → [IP Control] → [Pre-Shared Key] → [XXXX]
  5. Give your TV a static IP address, or make a DHCP reservation for a specific IP address in your router.
  6. Get/check IP address of your TV:
    [HOME] → [Settings] → [Network] → [Advanced settings] → [IP address]

Basic set-up steps

  1. Open app
  2. Add and register new control from main navigation menu:
    • Enter IP address of your TV
    • Define nick- and device names (used for display purposes)
    • Enter pre-shared key as defined in preparation of your TV (if option exists for your TV)
    • Enter challenge code as displayed on TV screen on registration (only of no pre-shared key is defined)
    • Watch for messages on your TV screen or within the app to handle errors accordingly
  3. For use as TV Browser plugin
    1. Open TV Browser app
    2. Register/enable this app as TV Browser plugin
    3. Open plugin settings from TV Browser
    4. Map TV Browser channel names to TV programs (mostly automated)
