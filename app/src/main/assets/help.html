Sony TV Switch

Main features

  • Works for many smart Sony TVs with network connection (model year ~2013 to ???)
  • Controls the Sony TV from your Android device via network (Wifi)
  • Lists the TV programs from the TV for easy switching
  • As TV Browser plugin: Switch to a TV program directly from the program guide (EPG)

Set-up steps

  1. Install and open app
  2. Add control
  3. Manage control
    1. Register control for your Sony TV
    2. Request program list from TV
  4. For use as TV Browser plugin:
    1. Open TV Browser app
    2. Register/enable this app as TV Browser plugin
    3. Open plugin settings from TV Browser
    4. Map channel names to TV programs

Further information is provided in the online help
